# Claude Code Instructions

작업을 시작하기 전에 반드시 다음 문서를 순서대로 읽는다.

1. `docs/PLAN.md`
2. `docs/GAME_DESIGN.md`
3. `docs/ART_DIRECTION.md`
4. `docs/PROJECT_CHECKPOINT.md`
5. `docs/STATUS.md`
6. `docs/HANDOFF.md`

적·보스 모션, 전체 3막 노드, 카드 UI, 자동 저장, 일반 적 P/T/Z 3종, 포션 2종, 공용 카드, 드로우·에너지·힘·소멸, 희귀도 기반 혼합 상점, 포션 전투 피드백, 런 덱 보기 UI까지 완료됐다. 현재 각 막은 10층이다: 1·2층 전투 → 3층 이벤트 → 4층 기존 적 2체전 → 5층 {상점,엘리트} → 6층 기존 적 2체전 → 7층 보물 → 8층 기존 적 2체전 → 9층 휴식 → 10층 보스. 2026-07-01 기준 활성 캐릭터 20종(T6·P5·Z9)·적 15종은 개별 라이브 텍스처를 사용한다. `pado`와 `peanut`은 게임에서 삭제됐다. 1·2·3막 적은 모두 전용 6포즈 시트를 사용한다. 활성 캐릭터 20종의 시그니처 카드와 기존 저장 마이그레이션이 완료됐다. 전체 116 테스트가 통과한다.

활성 카드 아트 60종(공용 40·시그니처 20)은
`assets/game/cards/{cardId}.png`에 연결돼 있다. 최종 검수·보정·매핑 기록은
`assets/review/cards-abc-v1/`에 있다.

향후 신규 이미지는 직접 생성하지 않는다. 필요한 콘셉트·파일명·크기·형식·
방향·연결 ID만 `docs/IMAGE_REQUESTS.md`에 기록하고 게임 구성 완성을 우선한다.

전투 후속 규칙: 지두두 idle은 첫 프레임 고정이며 다른 캐릭터도 정적 대기
컷아웃을 사용한다. 활성 20종은 모두 전용 25프레임 공격 시트를 사용하고,
T는 총격, P/Z는 전진 칼질·종족색 참격이다. 플레이어 실루엣 높이는 지두두
기준 208.38px다. 잔해 총잡이는 실제 좌향 6포즈 시트를 사용하며 런타임
반전을 사용하지 않는다.
