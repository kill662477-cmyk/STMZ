검증된 기반은 전체 1막 노드, 자동 저장, 일반 적 P/T/Z 3종, 포션 2종, 공용 카드 16종, 드로우·에너지·힘·소멸, 희귀도 기반 카드4·유물2·포션2 혼합 상점, 포션 전투 피드백, 런 덱 보기, 다막 프레임워크 + 2막 + 3막(적 5종 + 최종 보스 성간 전함), 3막 풀런 밸런스 QA 통과(막 전환 최소 체력 50% 보장)까지 연결됐다. 2026-07-01 기준 활성 캐릭터는 20종(T6·P5·Z9), 적은 15종이며 개별 라이브 텍스처를 사용한다. `pado`와 `peanut`은 게임에서 삭제됐다. 1·2·3막 적은 모두 전용 6포즈 시트를 사용한다. 활성 캐릭터 20종은 각자 시그니처 카드 1장을 시작 덱에 사용하며 기존 저장도 자동 마이그레이션된다. 전체 112 테스트가 통과한다. 3막 적 5종은 생성·검수·라이브 승격·`?combatQa=1` 화면 QA까지 완료됐다. Codex 재시작 후 전역 개인 플러그인 `$studio`를 시작점으로 사용할 수 있다.

활성 카드 아트는 총 60종(공용 40·시그니처 20)이며
`assets/game/cards/{cardId}.png`에 있다. 검수·보정·매핑 기록은
`assets/review/cards-abc-v1/`을 기준으로 한다.

모든 막은 10층이며 4·6·8층에서 기존 일반 적 2체가 동시에 등장한다.
새 적이나 이미지를 만들지 않고 기존 전투 에셋을 재사용한다. 향후 이미지가
필요하면 직접 생성하지 말고 `docs/IMAGE_REQUESTS.md`에 콘셉트·파일명·크기·
형식·방향·연결 ID를 남긴 뒤 게임 구성 구현을 계속한다.

전투 후속 규칙: 지두두 idle은 첫 프레임 고정이며 다른 캐릭터도 정적 대기
컷아웃을 사용한다. 활성 20종은 모두 전용 25프레임 공격 시트를 사용하고,
T는 총격, P/Z는 전진 칼질·종족색 참격이다. 플레이어 실루엣 높이는 지두두
기준 208.38px다. 잔해 총잡이는 실제 좌향 6포즈 시트를 사용하며 런타임
반전을 사용하지 않는다.

- 현재 중단 지점을 보존한다.
- 미검수 생성물을 라이브 에셋으로 승격하지 않는다.
- 사용자 승인 없이 다음 대량 생성 단계로 넘어가지 않는다.
- 작업 후 `docs/STATUS.md`와 `docs/HANDOFF.md`를 갱신한다.
