# SLAY THE MONSTARZ — 게임 디자인 (v2)

- 작성일: 2026-06-27
- 출처: v1 `GAME_DESIGN_SPEC`의 **게임플레이 설계**를 그대로 계승 (v1에서 문제였던 건 아트·레이아웃·실행이지 게임플레이 설계가 아니었음).
- 아트·레이아웃 방향은 `PLAN.md` 참조. 이 문서는 **규칙·구조·경제**만 다룬다.

## 1. 한 문장

20명의 MONSTARZ 캐릭터 중 하나를 골라 P/T/Z 카드로 덱을 키우고, 적의 의도를 읽으며 3막의 보스를 잡는 20~30분짜리 턴제 로그라이크 덱빌딩.

## 2. 핵심 설계 원칙

1. 한 런 평균 30~40분.
2. 규칙은 가볍게, 카드 선택엔 분명한 손익.
3. 적 행동은 예측 가능 — 매 턴 공격/방어 판단.
4. 캐릭터 개성은 고유 카드·유물·연출에 집중.
5. 연출은 전투 시간을 늘리지 않고 타격감만 강화.
6. MONSTARZ SURVIVORS 시스템·콘텐츠 미사용.

## 3. 런 흐름 / 승패

```
타이틀 → 캐릭터 선택 → 1막 지도 → 노드9 → 1막 보스 → 핵심보상
       → 2막 지도 → 노드9 → 2막 보스 → 핵심보상
       → 3막 지도 → 노드9 → 최종 보스 → 승리
```

- 승리: 3막 보스 처치.
- 패배: 플레이어 HP 0. 프로토타입은 기본 부활 없음(유물 1회성 예외만 허용).
- 자동 저장: 전투·노드 사이. 전투 중 새로고침 시 해당 전투 시작 상태로 복원.

## 4. 플레이타임 예산

| 항목 | 횟수 | 1회 | 합계 |
|---|--:|--:|--:|
| 일반 전투 | 15 | 40~90초 | 12~18분 |
| 엘리트 | 2~3 | 60~90초 | 3~4분 |
| 보스 | 3 | 90~150초 | 5~7분 |
| 이벤트·상점·휴식·보물 | 12 | 15~40초 | 5~7분 |
| 지도·보상·덱 | 전체 | - | 3~4분 |
| 총합 | 30노드 | - | 30~40분 |

## 5. 지도 규칙

- 3막, 각 막 일반 9층 + 보스 층. 층마다 2~3노드, 다음 층 1~2노드로 분기.
- 보스 직전 제외, 단일 경로가 2층 연속 안 됨. 도달 불가 노드는 흐리게. 보스는 막 진입 시 공개.

| 층 | 허용 노드 |
|---|---|
| 1 | 일반 전투 고정 |
| 2 | 일반 전투 고정 |
| 3 | 이벤트 고정 |
| 4 | 기존 일반 적 2체전 고정 |
| 5 | 상점 또는 엘리트 |
| 6 | 기존 일반 적 2체전 고정 |
| 7 | 보물 고정 |
| 8 | 기존 일반 적 2체전 고정 |
| 9 | 휴식 고정 |
| 10 | 보스 고정 |

노드 종류: 일반전투 / 엘리트(유물 확정) / 이벤트 / 상점 / 보물 / 휴식 / 보스.
생성 제약: 2체전은 해당 막 기존 일반 적 중 서로 다른 두 종류를 사용한다.
상점·엘리트는 5층에서만 선택하고 모든 경로에서 보스 직전 9층 휴식이 가능하다.

## 6. 전투 기본 규칙

플레이어 기본값: 최대 HP 캐릭터별 64~76 / 에너지 3 / 턴 드로우 5 / 손패 최대 10 / 시작덱 10(완성 16~22) / 포션 2슬롯.

카드 구역: 드로우·손패·버린·소멸. 드로우 더미 비면 버린 더미 섞어 보충.
카드 유형: 공격 / 스킬 / 파워 / 상태이상 / 저주 / 중립.

플레이어 턴: 방어도 제거(유지효과 적용) → 턴시작 지속효과 → 에너지 초기화 → 5장 드로우 → 카드 사용 → 턴 종료 → 손패 버림(보존 제외) → 턴종료 효과.
적 턴: 적 방어도 제거 → 지속효과 → 앞쪽 적부터 행동 → 사망 적 행동 취소 → 턴종료 효과 → 다음 의도 표시.

전투 종료: 모든 적 HP 0 즉시 승리(잔여 행동 미실행). HP는 다음 전투로 이어지고 방어도·전투전용 효과는 초기화.

## 7. 피해와 상태효과

피해 처리: 기본피해 → 공격력 증감 → 약화/취약 → 방어도 흡수 → HP 적용(최종 내림). 다단공격은 타격마다 방어도 별도 감소. HP 직접감소는 방어도 무시(카드 명시).

상태효과 8종 한정:

| 상태 | 효과 |
|---|---|
| 공격력 | 공격 피해 ± |
| 약화 | 주는 공격 피해 -25% |
| 취약 | 받는 공격 피해 +50% |
| 취약 방어 | 카드 방어도 -25% |
| 중독 | 턴 시작 수치만큼 HP 감소 후 1 감소 |
| 재생 | 턴 종료 수치만큼 회복 후 1 감소 |
| 보호막 유지 | 이번 턴 종료 시 방어도 유지 |
| 기절 | 예정 행동 1회 취소 후 해제 |

보스는 기절 완전면역 대신 행동 지연/피해 감소로 반응 가능.

## 8. P/T/Z 정체성 (전용 게이지 없이 공통 규칙 조합)

- **P 보호·충전**: 높은 방어도, 보호막 유지, 에너지 저장, 강한 단일 공격. 약점=저비용 공격 부족.
- **T 순환·화력**: 드로우, 버리기, 다단공격, 상황 대응. 약점=손패 순환 끊기면 약함.
- **Z 증식·대가**: 저비용 카드 생성, 중독, HP를 자원으로. 약점=장기전 체력관리·덱 비대.

## 9. 캐릭터 차별화

20명, 소속 종족 공용 카드풀 사용 + 다음으로 차별화: 고유 시작 유물 1 / 시그니처 카드 1 / 공용 기본카드 2장 교체·변형 / 고유 전신·얼굴 / 고유 시그니처 모션·VFX. 캐릭터별 독립 대형 카드풀·전용 게이지 금지.

## 10. 보상과 경제

- 골드: 시작 75, 일반 18~28, 엘리트 35~45, 이벤트 가변.
- 상점: 일반카드 40~60 / 고급 70~95 / 희귀 110~145 / 일반유물 120~160 / 고급유물 170~220 / 희귀유물 230~280 / 포션 45~80 / 첫 카드제거 75(+25씩). 진열 카드4·유물2·포션2.
- 휴식: 최대 HP 25% 회복 또는 카드 1장 강화.
- 보스 보상: 1·2막 희귀카드 3중1 + 핵심유물 3중1, 3막 즉시 승리.

## 13. 에셋 제작 방식 (확정)

- **AetherForge 미사용**: 신규 마스터·모션은 `GPT Image` 또는 `Nano Banana / Nano Banana Pro`로 직접 생성한다.
- **마스터 이미지(정적 원화)**: 승인 레퍼런스를 넣어 캐릭터 정체성과 전투 방향을 고정한다.
- **동작·애니메이션**: 캐릭터당 6개 안팎의 핵심 포즈를 보드 단위로 생성하고 `tools/keypose_board.py`로 자동 분리·크로마 제거·경계 QA한다. 실패 셀만 개별 보수한다.
- 단순 호흡·부유·넉백·사망은 Pixi 트윈으로 재사용하고, 실루엣이 변하는 공격·보스 포즈에 생성 이미지를 집중한다.
- 후보는 `assets/review/`에 두고 승인분만 게임 에셋 폴더로 승격한다.
- 캐릭터 방향: 카드=정면, 인게임 전투=측면(플레이어→오른쪽, 적→왼쪽). 정면 금지.

## 14. MVP / 빌드 순서 (확정)

단계별로 간다. 풀게임을 한 번에 만들지 않는다.

1. **1차 — 전투 수직 슬라이스**: 캐릭터 1명 + 일반 적 1 + 보스 1. 슬더스식 전투 씬(React+Pixi)으로 룩·타격감·아트 검증 우선. 맵·상점 없음.
2. **2차 — 1막 풀루프 (MVP, 첫 플레이 가능본)**: + 적 3종·엘리트 1 + 지도·카드보상·휴식·상점·이벤트, 카드 12~16·유물 5·포션 2, 1막 7층.
3. **3차+**: 2·3막 콘텐츠, 캐릭터 확장(20명 목표).
