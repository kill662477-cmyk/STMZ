# SLAY THE MONSTARZ — 재시작 기획 (v2)

- 작성일: 2026-06-27
- 최종 갱신: 2026-06-28
- 상태: 1차 전투 수직 슬라이스 완료, 2차 1막 풀루프의 런 엔진 기반 작성·검증 후 UI 통합 직전
- 이 문서는 v1 폴더 전체 삭제 후 **처음부터 다시** 잡는 기획이다.

## 0. 왜 다시 시작하는가 (v1 실패 교훈)

v1은 다음 3가지가 핵심적으로 잘못됐다. v2는 이걸 규칙으로 못 박는다.

1. **아트 정체성 무시** — 사용자가 준 레퍼런스(치비 3D + FUT 네온 카드 + MONSTARZ 저지)를 버리고 무국적 SF 전사를 생성. 원작(슬더스)과도 안 닮음.
2. **레이아웃이 박스 대시보드** — 캐릭터를 테두리 패널 박스에 가둠. 슬더스는 풀배경 전장에 캐릭터가 자유롭게 서 있다.
3. **품질을 멋대로 "합격" 처리** — "기술적으로 파일이 나옴"을 "결과가 좋음"으로 포장. 판단은 사용자가 한다.

## 1. 목표 (GOAL)

> **Slay the Spire 원작 분위기 90% 싱크로율** + MONSTARZ 정체성(멤버·저지·스타크래프트 종족)을 입힌 턴제 로그라이크 덱빌딩 웹게임.

- 1순위: 슬더스의 톤·전투감·화면 구성 90% 재현.
- 절대 조건: MONSTARZ 정체성을 무시하지 않는다.

## 2. 확정된 방향 (DECIDED)

- **아트 톤**: 슬더스 톤 우선. 캐릭터를 슬더스식 손그림 다크판타지로 재해석하되 얼굴·멤버·종족으로 MONSTARZ 정체성 유지.
- **레이아웃**: 슬더스식 풀배경 전장. 플레이어 왼쪽 / 적 오른쪽 / 발밑 HP / 하단 부채꼴 손패 / 좌하단 에너지 오브 / 구석 드로우·버린 더미. **캐릭터를 박스에 가두지 않는다.**
- **캐릭터 방향(중요)**: 슬더스 전투화면은 **정면 응시가 아니다.** 플레이어는 화면 왼쪽에서 **오른쪽(적)을 바라보는 측면/3-4 뷰**, 적은 **왼쪽(플레이어)을 바라봄.** 서로 마주보는 구도.
  - **카드/로스터 아트** = 정면 (기존 FUT 카드처럼)
  - **인게임 전투 스프라이트** = 측면 지향 (플레이어→오른쪽, 적→왼쪽). 정면 금지.
- **장르**: 턴제 로그라이크 덱빌딩 (실시간 액션 아님).
- **별개 프로젝트**: MONSTARZ SURVIVORS의 시스템·콘텐츠를 가져오지 않는다.
