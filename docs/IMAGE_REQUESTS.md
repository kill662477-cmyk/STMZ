# SLAY THE MONSTARZ — 외부 이미지 요청 목록

## 제작 원칙

- 게임 규칙·콘텐츠·진행·UI·밸런스 구현을 우선한다.
- Codex는 앞으로 필요한 이미지를 직접 대량 생성하지 않는다.
- 이미지가 없어도 임시 색상·기존 에셋·절차적 연출로 기능 구현을 계속한다.
- 꼭 필요한 신규 이미지만 아래 양식으로 사양을 남기고 사용자가 외부에서 제작한다.
- 전달받은 이미지는 ID·크기·알파·여백·중복을 검수한 뒤 라이브에 연결한다.

## 이미지 요청 양식

각 요청에는 아래 항목을 빠짐없이 적는다.

- 요청 ID / 우선순위 / 상태
- 용도와 노출 화면
- 파일명과 최종 저장 경로
- 픽셀 크기, 프레임 배열, 파일 형식, 알파 필요 여부
- 구도·시점·방향·실루엣
- 종족·막·색상·조명 콘셉트
- 포함 요소 / 금지 요소
- 필요한 변형 수
- 연결할 코드 또는 콘텐츠 ID

## 현재 요청 큐

없음. 2026-07-01의 10층 지도·기존 적 2체전 확장에는 신규 이미지가 필요하지 않다.
