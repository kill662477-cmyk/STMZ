## 2026-07-01 — 전 막 10층 확장·기존 적 2체전

- 1·2·3막 지도를 동일한 10층 진행으로 확장했다:
  `전투 → 전투 → 이벤트 → 2체전 → 상점/엘리트 → 2체전 → 보물 → 2체전 → 휴식 → 보스`.
- 4·6·8층은 각 막의 기존 일반 적 3종을 두 종류씩 조합한다.
  신규 적 데이터·스프라이트·이미지는 추가하지 않았다.
- `BattleEngine`이 적 배열과 집중 대상을 관리한다. 공격은 앞의 생존 적을
  먼저 노리고, 적 턴에는 살아 있는 두 적이 차례로 모두 행동한다.
  첫 적을 쓰러뜨리면 다음 적으로 자동 전환하며 모든 적을 쓰러뜨려야 승리한다.
- `BattleStage`와 전투 HUD가 적 2체의 스프라이트·HP·방어도·상태·의도를
  각각 렌더링하고 피해·공격·방어·사망 연출을 적 인덱스별로 재생한다.
- 기존 7층 저장 노드 ID는 의미를 유지하고 `f3b`·`f4b`·`f5b` 전투 노드만
  삽입해 구형 저장이 상점·보물·휴식·보스를 다른 종류로 복원하지 않게 했다.
- 10층 지도가 한 화면에 들어오도록 간격과 노드 크기를 조정하고 진행 표시를
  고정 `7`에서 실제 층 수로 변경했다.
- 자동 밸런스: 1막 안전 경로 8시드 중 5승, 전체 3막 16회 중 1회 완주.
  이전보다 확실히 어렵지만 전 구간 클리어 가능성은 유지한다.
- 실제 브라우저에서 10층 지도와 4층 잔해 총잡이+부유 파수병 2체전,
  두 적의 연속 행동을 확인했다. 신규 콘솔 오류·경고 0.
- 검증: typecheck PASS, **10파일 116테스트 PASS**, production build PASS.
