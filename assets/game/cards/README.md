# 승인 카드 아트

사용자 승인까지 끝난 카드 일러스트만 `{cardId}.png` 이름으로 이 폴더에 둔다.

- 공용 카드는 캐릭터별로 만들지 않는다. 같은 `{cardId}`를 쓰는 모든 캐릭터가 이미지 1장을 공유한다.
- 캐릭터 전용 이미지는 별도 `cardId`를 가진 시그니처 카드만 사용한다.
- 강화 카드 `{cardId}+`는 기본 카드 아트를 자동 재사용한다.
- 원본 이미지에는 카드 프레임, UI, 글자, 숫자, 로고를 넣지 않는다.
- 후보는 먼저 `assets/review/cards/`에서 검수한다.

2026-07-01 기준 활성 카드 60종(공용 40 + 시그니처 20)의 전용 아트가
연결됐다. 현재 배치의 검수 기록과 최종 후보는
`assets/review/cards-abc-v1/`에 있다.
