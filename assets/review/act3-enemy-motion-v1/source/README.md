# Source boards

Generated 2×3 chroma-key boards are stored here before frame extraction and QA.

- Green boards are processed directly with `tools/keypose_board.py`.
- `abyssal_cluster-v1.png` and `abyssal_cluster-v2.png` use magenta chroma to
  preserve the creature's yellow-green acid details.
- `abyssal_cluster-v1-alpha.png` and `abyssal_cluster-v2-alpha.png` are the
  transparent intermediates used for frame extraction.
