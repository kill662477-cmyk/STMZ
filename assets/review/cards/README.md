# 카드 아트 검토 대기열

카드 일러스트 후보와 검수 기록을 보관한다. 사용자 승인 전에는 `assets/game/cards/`로 승격하지 않는다.

## 제작 단위

- 공용 카드는 캐릭터별 이미지가 아니라 `{cardId}`별 공용 이미지 1장을 사용한다.
- 같은 카드를 쓰는 모든 캐릭터가 같은 이미지를 공유한다.
- 캐릭터 식별 이미지는 별도 `cardId`를 가진 시그니처 카드에만 허용한다.

## 현재 공용 앵커

- 카드: `strike` / 타격
- 범위: T 종족 공용 기본 카드
- 프롬프트: `tools/card_art_prompts.json`
- 상태: 캐릭터가 드러나지 않는 무기·충격 모티프의 새 후보 생성 대기

## 폐기 후보

- `rejected/strike-jdd-character-specific-v1.png`
- 폐기 사유: 공용 기본 카드에 지두두의 얼굴·헤어가 들어가 캐릭터별 제작을 유발하며, 표현도 목표보다 3D 렌더에 가깝다.
